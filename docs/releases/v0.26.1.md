# v0.26.1 — Soul Mode English Integration Copy

**Date:** 2026-06-10

## Highlights

- Replaces Portuguese hard-coded Soul Mode integration prompts with English baseline copy.
- Updates additive identity integration section headings to English defaults.
- Aligns CV19 docs, release notes, and focused Soul Mode tests with the English baseline.
- Preserves the v0.26.0 Soul Mode Integration behavior and scope.

## Where We Started

A live Soul Mode test conducted in English exposed several Portuguese prompts in
confirmation and integration flow. The issue was not localization, because Mirror
Mind does not have an i18n layer yet. The problem was simpler: framework-level
hard-coded copy had been written in Portuguese inside an English-baseline
runtime.

## What Changed

The Pi Soul Mode skill now asks closing, integration, confirmation, and
post-apply questions in English. The layer-language examples in the skill were
also translated so future proposals are guided by the same baseline language.

The additive identity integration headings written by `memory soul apply` now use
English section titles:

```text
self    → ## New Incorporated Principles
shadow  → ## New Hidden Needs Recognized
ego     → ## New Operational Patterns Identified
persona → ## New Participation Patterns Revealed
```

The CV19 story docs, tests, and v0.26.0 release note were reconciled so the
record no longer teaches Portuguese as the framework default.

## Conscious Exclusions

- No localization system.
- No dynamic language detection.
- No migration of previously written user identity content.
- No change to Soul Mode ritual behavior beyond baseline copy and section headings.

## What We Learned

When localization is absent, the framework still needs one clear base language.
For Mirror Mind internals and hard-coded runtime contracts, that language is
English. User-authored material may remain in any language, but product copy and
runtime defaults should not drift into mixed-language behavior.

## Next Horizon

Future localization can make Soul Mode speak in the user's preferred language.
Until then, hard-coded Mirror Mind surfaces should remain consistently English.
