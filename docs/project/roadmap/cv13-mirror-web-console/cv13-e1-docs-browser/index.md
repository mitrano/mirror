# Historical note — Docs Browser

**Status:** Superseded by CV9.E6 Web Visibility

This folder preserves an older CV13 planning slice for a local docs browser. The docs browser direction was absorbed by the broader web foundation and visibility work completed in CV9.E6, so this is no longer an active CV13 story and should not be used as the current CV13.E1.

Current CV13.E1 is **Read-only Web Refinement**, described in the [CV13 Mirror Web Experience index](../index.md).

---

## Original intent

The original slice proposed a local read-only web console that could browse Mirror documentation and render Markdown as HTML through:

```text
GET /api/docs/tree
GET /api/docs/file?path=...
```

That capability is now part of the broader web surface history rather than the next roadmap front door.
