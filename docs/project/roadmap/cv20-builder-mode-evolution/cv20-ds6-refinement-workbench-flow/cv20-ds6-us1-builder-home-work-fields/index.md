# CV20.DS6.US1 — Builder Home Work Fields

**Type:** User Story  
**Status:** 🟡 Planned

## Story

As Navigator, I want Builder activation to show both Delivery and Refinement
fields so that I can choose whether to continue roadmap delivery, work from the
Workbench, or organize refinement material before execution.

## Outcome

Builder load renders a situated home surface for Ariad-adopted journeys with
current roadmap state, delivery cursor state, Workbench summary, active RS when
present, and available next moves.

## Notes

This story should preserve the activation boundary: loading Builder is
orientation, not execution consent.
