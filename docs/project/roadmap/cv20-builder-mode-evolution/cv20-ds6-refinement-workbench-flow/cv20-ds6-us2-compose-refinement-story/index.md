# CV20.DS6.US2 — Compose A Refinement Story

**Type:** User Story  
**Status:** 🟡 Planned

## Story

As Navigator, I want to create a Refinement Story and add Change Requests to it
while I dogfood a product flow so that observed adjustments are preserved as
requested changes without becoming roadmap delivery work.

## Outcome

Builder can create or open an RS, capture CRs into it, render `Change Request
Captured` surfaces, and render a `Refinement Story Overview` surface showing the
CRs associated with the RS.

## Notes

This story does not implement automatic clustering or Signal Field behavior.
