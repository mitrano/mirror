# CV20.DS6.US3 — Pull A Refinement Story

**Type:** User Story  
**Status:** 🟡 Planned

## Story

As Navigator, I want to pull a Refinement Story from the Workbench so that
Builder enters Refinement Work with an active RS instead of assuming roadmap
Delivery Work.

## Outcome

Builder records the active RS, renders a `Refinement Story Pulled` surface, and
resumes the RS on later Builder activation.

## Notes

The source field determines the flow: roadmap items enter Delivery Work,
Workbench items enter Refinement Work.
