# Método de estudo pessoal

Pasta associada à jornada `metodo-estudo-pessoal`.

## Objetivo

Desenvolver um método simples, recorrente e concluível para estudar temas de interesse profissional, intelectual ou de crescimento pessoal.

## Arquivos

- `source/metodo-estudo-gemini.md` — proposta recebida do Gemini.
- `source/metodo-estudo-grok.md` — proposta recebida do Grok.
- `source/metodo-estudo-claude.md` — proposta recebida do Claude.
- `source/metodo-estudo-gpt.md` — proposta recebida do GPT.
- `source/eca-complexidade-nivel-2-gpt.md` — aplicação do Método ECA ao estudo de complexidade no Nível 2, recebida do GPT.
- `source/eca-complexidade-nivel-3-gpt.md` — aplicação do Método ECA ao estudo de complexidade no Nível 3, recebida do GPT.
- `metodo-proposto-mirror.md` — proposta inicial feita pelo Mirror nesta jornada.
- `analysis/comparativo-metodos-estudo.md` — comparação entre as propostas, com prós, contras e recomendação inicial.
- `templates/template-exploracao-inicial.md` — modelo curto para triagem de novos temas.
- `templates/template-compreensao-funcional.md` — modelo de Nível 2 para síntese funcional.
- `templates/template-dominio-progressivo.md` — modelo de Nível 3 para ciclos de domínio progressivo.
