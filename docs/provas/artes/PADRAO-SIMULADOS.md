# Padrão Operacional — Simulados de Artes do 7º ano

Este arquivo complementa o padrão geral de simulados registrado em:

```text
docs/provas/PADRAO-SIMULADOS-GERAL.md
```

Ele registra o padrão específico para **Artes / História da Arte, 7º ano, provas discursivas**.

Este padrão descreve **como Artes costuma perguntar**. Ele não fixa o conteúdo de uma prova específica. O conteúdo de cada simulado deve vir da revisão atual do bimestre/prova em questão.

---

## 1. Regra central contra contaminação de conteúdo

Para gerar simulados de Artes:

> A estrutura vem das provas reais/análises anteriores; o conteúdo vem da revisão atual.

Isso significa:

- provas anteriores servem para imitar formato, nível, comandos, tipos de questão e lógica avaliativa;
- a revisão atual define os temas, artistas, obras, movimentos e conceitos que devem aparecer no simulado;
- não reutilizar automaticamente artistas ou temas da P1 em P2, P3 etc., a menos que eles também apareçam na revisão atual.

---

## 2. Padrão avaliativo identificado em Artes

A prova de Artes segue a lógica:

> obra/imagem + conceito estudado + pergunta curta de identificação, interpretação, vocabulário ou criação visual.

Diferente de Ciências, que costuma usar situações cotidianas, Artes usa principalmente:

- obras de arte;
- artistas;
- movimentos artísticos;
- técnicas;
- símbolos;
- características visuais;
- leitura/interpretação de imagem;
- produção artística.

---

## 3. Tipos de questão recorrentes

Os simulados de Artes devem misturar, quando compatível com a revisão atual:

1. **Citação de características**
   - Exemplo de comando: “Cite duas características...”

2. **Memorização histórica ou conceitual**
   - nomes de grupos, movimentos, técnicas, artistas ou períodos.

3. **Interpretação simbólica**
   - explicar o significado de elementos visuais em uma obra.

4. **Justificativa conceitual**
   - justificar por que uma obra/artista se aproxima de um movimento ou conceito.

5. **Identificação técnica**
   - reconhecer técnica a partir de descrição ou imagem.

6. **Completar lacunas com banco de palavras**
   - vocabulário artístico, nomes de movimentos, artistas, técnicas ou conceitos.

7. **Análise visual**
   - reconhecer características visuais, composição, profundidade, cor, distorção, padrão etc.

8. **Produção artística**
   - criar uma composição visual com critérios claros.

---

## 4. Questão prática/criativa

Quando a prova real da disciplina incluir produção artística, os simulados devem preservar esse tipo de questão.

A questão criativa deve trazer:

- tema ou conceito a representar;
- restrições claras;
- orientação de uso do espaço;
- pedido de cor/capricho quando compatível;
- critério de avaliação no gabarito comentado.

O gabarito da questão criativa não deve indicar uma única resposta correta. Deve indicar critérios de avaliação.

---

## 5. Questão de excelência

A questão de excelência em Artes deve exigir aplicação ou integração de conceitos visuais.

Ela deve preferencialmente:

1. partir de uma obra ou descrição visual;
2. pedir análise, comparação ou aplicação de conceito;
3. exigir justificativa;
4. avaliar leitura visual, não apenas memorização.

Se a prova real usar uma numeração especial — por exemplo, Questão de Excelência correspondendo à Questão 07 — preservar esse padrão nos simulados da mesma família de prova.

---

## 6. Uso de imagens

Quando imagens reais não estiverem disponíveis no simulado:

- descrever a obra ou cena de forma clara;
- informar que o aluno deve imaginar/observar a imagem descrita;
- não depender de uma imagem ausente para uma resposta impossível.

Quando imagens forem usadas em `.docx`, garantir boa diagramação.

---

## 7. Gabarito comentado em Artes

Para cada questão, usar:

```text
Resposta esperada:
[resposta modelo]

Comentário:
[critério simples de correção]
```

Para questões interpretativas, aceitar respostas equivalentes quando o conceito estiver correto.

Para questões criativas, avaliar:

- relação com o tema;
- respeito às restrições;
- uso do espaço;
- uso de cor, quando solicitado;
- capricho/apresentação;
- originalidade ou solução visual coerente.
