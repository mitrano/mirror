# Prompt Gerador — Simulado de Geografia P1

Este prompt é uma **instância específica da P1 de Geografia**.

Ele usa:

- a revisão da P1 como fonte de conteúdo;
- a prova real de P1 como referência estrutural;
- o padrão específico de Geografia como guia de formato.

Atenção: este arquivo contém conteúdo da P1. Para P2, P3 etc., criar nova instância substituindo integralmente o bloco de conteúdo pela revisão atual.

---

## Prompt

Você é um professor especialista em Geografia do 7º ano. Gere um simulado de Geografia P1 para um aluno de 12 anos.

O simulado deve imitar o padrão da prova real analisada, mas com questões inéditas.

## Fontes

Use como referência estrutural:

```text
docs/provas/geografia/p1/QUESTOES-EXTRAIDAS.md
docs/provas/geografia/p1/ANALISE-PROVA-REVISAO.md
docs/provas/geografia/PADRAO-SIMULADOS.md
```

Use como fonte principal de conteúdo:

```text
docs/provas/geografia/p1/revisoes/P1 - GEOGRAFIA - v02.txt
```

## Conteúdos da revisão P1

O simulado deve trabalhar os conteúdos da revisão:

1. Estado;
2. nação;
3. Estado-nação;
4. macrorregiões brasileiras do IBGE;
5. características das regiões Norte, Nordeste, Centro-Oeste, Sudeste e Sul;
6. formação/interiorização do território brasileiro;
7. atividades econômicas coloniais:
   - pau-brasil;
   - cana-de-açúcar;
   - pecuária;
   - ouro/mineração;
   - drogas do sertão;
   - café;
8. Constituição de 1988;
9. redemocratização;
10. direitos civis, políticos e sociais;
11. comércio triangular;
12. Europa, África e América/Brasil no comércio triangular;
13. Divisão Internacional do Trabalho;
14. papel histórico do Brasil como fornecedor de matérias-primas/produtos primários.

## Estrutura obrigatória

Criar um simulado com estrutura semelhante à prova real:

1. Questões discursivas.
2. Uso de notícia, trecho, publicação, mapa ou esquema como contexto.
3. Questões com comandos como explique, cite, indique e identifique.
4. Pelo menos uma questão com mapa ou descrição de mapa.
5. Pelo menos uma questão sobre regionalização/macrorregiões, se compatível com a revisão.
6. Pelo menos uma questão sobre formação territorial/economia colonial.
7. Pelo menos uma questão sobre Constituição/redemocratização ou organização política.
8. Pelo menos uma questão sobre comércio triangular ou DIT.
9. Questão de Excelência com interpretação de esquema/processo e síntese.
10. Gabarito comentado ao final.

## Regras de linguagem

- Linguagem adequada ao 7º ano.
- Perguntas curtas e claras.
- Não usar conteúdo fora da revisão P1.
- Usar linhas de resposta com exatamente 93 underscores:

```text
_____________________________________________________________________________________________
```

## Gabarito comentado

Ao final, incluir:

```text
# GABARITO COMENTADO
```

Para cada questão:

```text
## Questão XX

Resposta esperada:
[resposta modelo]

Comentário:
[critério simples de correção]
```

Para respostas discursivas, aceitar variações quando o conceito estiver correto.

## Critérios de qualidade

Antes de finalizar, verifique se:

1. O simulado parece uma prova real de Geografia do 7º ano.
2. A estrutura imita a prova P1 analisada.
3. O conteúdo vem da revisão P1.
4. Há fontes/contextos, mapa ou esquema.
5. O gabarito explica o critério de correção.
6. Não há conteúdo fora da revisão.
