# Prompt Gerador — Simulado de Filosofia P1

Este prompt é uma **instância específica da P1 de Filosofia**.

Ele usa:

- a revisão da P1 como fonte de conteúdo;
- a prova real de P1 como referência estrutural;
- o padrão específico de Filosofia como guia de formato.

Atenção: este arquivo contém conteúdo da P1. Para P2, P3 etc., criar nova instância substituindo integralmente o bloco de conteúdo pela revisão atual.

---

## Prompt

Você é um professor especialista em Filosofia e Cidadania do 7º ano. Gere um simulado de Filosofia P1 para um aluno de 12 anos.

O simulado deve imitar o padrão da prova real analisada, mas com questões inéditas.

## Conteúdos da revisão P1

O simulado deve trabalhar:

1. identidade brasileira;
2. formação cultural do Brasil;
3. pluralidade cultural;
4. crítica à ideia de Brasil homogêneo;
5. eurocentrismo;
6. apagamento das culturas indígenas e africanas;
7. violência da colonização;
8. problema do termo “índio”;
9. escravidão e apagamento cultural africano;
10. sincretismo religioso;
11. imigrações dos séculos XIX e XX;
12. características plurais do povo brasileiro.

## Estrutura obrigatória

Criar questões discursivas no padrão da prova real:

1. perguntas com explique, cite e justifique;
2. pelo menos uma questão de exemplo aplicado;
3. pelo menos uma questão com situação/charge descrita;
4. questão de excelência com síntese;
5. gabarito comentado;
6. formatação conforme `docs/provas/PADRAO-FORMATACAO-SIMULADOS.md`.
